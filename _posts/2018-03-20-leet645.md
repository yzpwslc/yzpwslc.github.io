<p>题目：集合缺失</p>
<p>分析：寻找1到N集合中错误的元素,首先通过计算差集寻找缺失元素，通过异或寻找出现两次元素</p>
<p>代码如下：</p>
{% highlight python %}
class Solution(object):
class Solution(object):
    def findErrorNums(self, nums):
        """
        :type nums: List[int]
        :rtype: List[int]
        """
        lose = list(set(range(1,len(nums) + 1)) - set(nums))
        lose.append(reduce(operator.xor,nums + lose + range(1,len(nums) + 1)))
        return lose
